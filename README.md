# TetherUSDBridgedZED20
"Tether USD Bridged ZED20 (USDT.z) adalah aset digital mutakhir di jaringan BNB Smart Chain yang mengedepankan efisiensi standar ZED20. Dengan total supply 27,5 Triliun, proyek ini bertujuan untuk menyediakan solusi bridging yang transparan, dapat ditarik kembali (withdrawable), dan sepenuhnya dapat dipertukarkan (swapable) dalam ekosistem DeFi."
